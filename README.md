# 🚀 StudySphere

Smart Student Workspace & Resource Hub built with the MERN stack.

StudySphere helps students organize study materials, manage assignments, collaborate in study groups, and share academic resources through a clean and modern dashboard experience.

---

# ✨ Features

- 🔐 JWT Authentication
- 📚 Subject & Workspace Management
- 📝 Notes & Resource Uploads
- 👥 Study Groups
- 📋 Assignment Tracker
- 💬 Discussion Threads
- 🔎 Smart Search & Filters
- 📊 Dashboard Analytics
- 🌙 Responsive Dark Mode UI

---

# 🚀 Getting Started

## Clone the Repository

```bash
git clone https://github.com/2ndjoy/studysphere-
cd studysphere
```

---

## Install Dependencies

### Frontend

```bash
cd client
npm install
```

### Backend

```bash
cd server
npm install
```

---

# 🔑 Environment Variables

Create a `.env` file inside the `server/` directory:

```env
PORT=5000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key

CLOUDINARY_CLOUD_NAME=your_cloud_name
CLOUDINARY_API_KEY=your_api_key
CLOUDINARY_API_SECRET=your_api_secret
```

---

# ▶️ Run the Application

## Start Backend

```bash
cd server
npm run dev
```

## Start Frontend

```bash
cd client
npm run dev
```
